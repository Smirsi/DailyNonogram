# Kurzinformation
Das Programm soll eine App für den iOS, ipadOS und macOS Appstore werden, mit der ein Bediener ein Nonogramm lösen kann. Jeden Tag ist ein neues Nonogramm verfügbar. Der Benutzer kann Werbung eingeblendet bekommen und kann Extra-Funktionen freischalten, wenn er für Premium bezahlt.
